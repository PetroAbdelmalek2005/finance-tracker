Google Sheets Connection
Your data syncs to a Google Sheet so it's available on all devices. Update your credentials here if needed.
How to get your Apps Script URL:
In your Google Sheet → Extensions → Apps Script → paste the script code → Deploy → New deployment → Web App → Execute as: Me → Who has access: Anyone → Deploy → copy the URL.

Google Sheets Sync
Manually push your local data up to Google Sheets, or pull fresh data down from Google Sheets to this device. Use Push if you've made changes locally that haven't synced. Use Pull if another device has newer data.
Data Backup
Export all your data as a JSON file. Save it to your OneDrive folder as a manual backup. To restore, use Import.
Import Bank Statement
Import transactions from your bank. Supports OFX/QFX files (TD, RBC, BMO, Scotiabank) and CSV files (Wealthsimple, EQ Bank). Duplicate transactions are automatically detected and skipped.
How to get your OFX file:
TD: Online Banking → Accounts → Download → Microsoft Money (.ofx)  ·  RBC: Online Banking → Download Transactions → OFX  ·  BMO: Online Banking → Export → OFX format
Danger Zone
Permanently delete all local data. This does not affect your Google Sheet.
