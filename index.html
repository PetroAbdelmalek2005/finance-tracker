Setup (2 minutes):
1. Go to github.com → Settings → Developer Settings → Personal access tokens → Tokens (classic)
2. Generate new token → check gist scope → copy the token
3. Go to gist.github.com → New gist → set to Secret → create → copy the Gist ID from the URL
Setup (2 minutes):
1. Sign up at jsonbin.io (free)
2. Dashboard → API Keys → copy your Secret Key
3. Dashboard → Bins → Create Bin → set name, make Private → copy the Bin ID
Setup: Google Sheet → Extensions → Apps Script → paste the script → Deploy → Web App → Execute as Me → Access: Anyone → copy URL.

Sync
Push sends your local data to the cloud. Pull replaces local data with what's in the cloud. Auto-sync does this automatically.
Data Backup
Export all your data as a JSON file. Save it to your OneDrive folder as a manual backup. To restore, use Import.
Import Bank Statement
Import transactions from your bank. Supports OFX/QFX files (TD, RBC, BMO, Scotiabank) and CSV files (Wealthsimple, EQ Bank). Duplicate transactions are automatically detected and skipped.
How to get your OFX file:
TD: Online Banking → Accounts → Download → Microsoft Money (.ofx)  ·  RBC: Online Banking → Download Transactions → OFX  ·  BMO: Online Banking → Export → OFX format
Danger Zone
Permanently delete all local data. This does not affect your Google Sheet.
